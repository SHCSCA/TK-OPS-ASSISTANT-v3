# Voice Segment Preview Design

## 范围

配音中心新增两种预览范围：

- `整段`：播放当前 VoiceTrack 的完整音频。
- `当前段落`：播放左侧当前选中的脚本段落对应音频区间。

本阶段不做每段单独 TTS，不新增多份段落音频文件，不改变现有生成任务与 Runtime 音频流接口。

## 数据流

`VoiceStudioPage` 将脚本段落列表、当前段落索引、选中音轨传给 `VoicePreviewStage`。`VoicePreviewStage` 在音频元数据加载后拿到真实总时长，并通过 `voice-preview-ranges.ts` 将段落估算时长按真实音频总时长缩放为段落播放区间。

## 交互

播放器默认处于 `整段` 模式。用户点击 `当前段落` 后，播放行为被限制在当前选中段落区间内；切换左侧段落时，若处于当前段落模式，播放器会跳到新段落起点。播放到段落终点后自动暂停。

## 异常处理

如果没有音频、没有段落或段落区间无法计算，则 `当前段落` 按钮禁用，播放器保留现有中文状态提示。音频加载失败继续显示中文错误。

## 测试

新增 helper 单元测试覆盖区间缩放与时间格式化；页面测试覆盖整段/当前段落切换和播放器 URL 不使用本地文件路径。
